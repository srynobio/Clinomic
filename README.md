#Clinomic

Clinomic allow clinical annoatation of GVF files using:

* ClinVar
* dbgap
* HGVS
* SNOMED CT
* LOINC
* NCBI
* refseq

##Installation

To install this module, run the following commands:
```
	perl Makefile.PL
	make
	make test
	make install
```
